Part 1 — IBKR account setup
1
Open IBKR account
Go to ibkr.com → Open Account. Select Individual. Choose "Margin" account type — required for short selling. Fund with minimum €8,000. Approval takes 1–3 business days.
2
Enable options trading
In IBKR Client Portal → Settings → Trading Experience → Options. Request Level 2 (covered calls + cash-secured puts). Level 3 for spreads. Usually approved same day.
3
Subscribe market data
Client Portal → Settings → Market Data Subscriptions. Subscribe to:
US Equity & Options (OPRA) US Securities Snapshot
Required for real-time options quotes and Greeks. ~$10/month.
4
Install IB Gateway
Download IB Gateway (not full TWS — lighter, headless-compatible) from ibkr.com/ibgateway. This is the API bridge your bot connects to.
ibgateway/bin/ibgateway & # start headless
5
Enable TWS API
Open IB Gateway GUI once. Go to: Configure → Settings → API → Settings.
✓ Enable ActiveX and Socket Clients
✓ Allow connections from localhost
Paper port: 7497  |  Live port: 7496
Part 2 — Server & bot deployment
6
Provision AWS EC2 server
Spin up a t3.small in us-east-1 (closest to IBKR servers). Ubuntu 22.04 LTS. Open inbound ports: 22 (SSH), 5000 (dashboard), 7497/7496 (localhost only).
aws ec2 run-instances --image-id ami-0c02fb55956c7d316 \ --instance-type t3.small --region us-east-1
7
Install dependencies
sudo apt update && sudo apt install -y python3-pip xvfb pip install ib_insync flask flask-socketio \ eventlet pytz nest_asyncio
8
Upload bot files via SCP
scp -r ./ibkr-compounder ubuntu@YOUR_IP:~/
9
Start IB Gateway headless (Xvfb)
Xvfb :1 -screen 0 1024x768x24 & DISPLAY=:1 ~/ibgateway/bin/ibgateway &
Login with IBKR credentials. Keep it running — systemd service recommended (see deploy/ibgateway.service).
10
Start bot (paper mode first!)
cd ~/ibkr-compounder/bot python main_bot.py # paper mode (port 7497) python main_bot.py --live # live mode (port 7496)
11
Start dashboard
cd ~/ibkr-compounder/dashboard python server.py &
Open http://YOUR_SERVER_IP:5000 in your browser. Dashboard auto-refreshes every 2 seconds via WebSocket.
12
Auto-start with systemd
sudo cp deploy/bot.service /etc/systemd/system/ sudo cp deploy/dashboard.service /etc/systemd/system/ sudo systemctl enable bot dashboard sudo systemctl start bot dashboard
